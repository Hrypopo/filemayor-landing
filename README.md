## ⚡ What is FileMayor?

FileMayor is an extremely fast, offline-first application designed to bring perfect order to your filesystem chaos. Built with a pure Node.js core and a pristine desktop interface, it intelligently scans, categorizes, and organizes your files into deterministic structures. 

*Zero cloud dependencies. Absolute privacy. Maximum psychological safety.*

### 🌟 Core Capabilities

- **Intelligent Organization:** Automatically sort thousands of files into precise categories (Code, Images, Archives, Documents, Configs, etc.).
- **Deterministic Rollback Journal:** Made a mistake? Press `Undo`. FileMayor logs every single move securely, giving you the psychological safety to organize aggressively.
- **Deep System Cleaning:** Safely purge system caches, temporary digital rot, and bloated `$TMPDIR` logs in milliseconds.
- **AI Standard Operating Procedures (SOPs):** Upload a simple `.md` file containing natural language instructions on *how* you want your files structured. FileMayor's Gemini-powered parser will deterministically map your arbitrary rules into concrete system actions.
- **Background Watcher (Pro):** Set up rules and let FileMayor monitor your `Downloads` or `Desktop` folders in real-time, instantly routing new files to their correct homes.

## 🚀 Installation & Download

FileMayor is available in two flavors:

### 1. The Desktop Application
For the complete visual experience, download the pre-compiled binaries for your system directly from our official website:

👉 **[Download FileMayor Desktop](https://filemayor.com)**

### 2. The CLI Core
For developers who prefer the raw power of the terminal, the pure `filemayor` engine can be installed globally via NPM:

```bash
npm install -g filemayor
```

*Required: Node.js ≥ 18*

## 🛡️ Security First

The FileMayor GUI isn't just a wrapper; it's a deeply hardened sandbox.
- **100% Local Execution:** Your files never leave your machine.
- **No Third-Party Analytics:** We don't track what you organize.
- **Path Validation:** Advanced IPC bridge preventing Local File Inclusion (LFI) and Path Traversals.
