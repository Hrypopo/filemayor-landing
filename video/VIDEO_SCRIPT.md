## PRODUCTION NOTES FOR AI TOOLS

### For Sora (OpenAI):
- Generate each scene as a separate 10-second clip
- Use **consistent camera language**: slow pushes, orbital movements, no fast cuts
- Specify "anamorphic lens, cinematic render, 24fps, film grain" in every prompt
- Stitch in CapCut or Premiere with 8-frame cross-dissolves between scenes

### For Kling (Kuaishou):
- Kling handles 3D floating objects well — scenes 1, 2, 4 will work best
- For scene 3 (card grid), add "product visualization style, commercial quality"
- Use "motion control camera" phrasing for the orbital shots

### For Runway Gen-3:
- Best for scene 5 (the undo reversal) — Runway handles reverse motion naturally
- For text overlays: generate video clean, add text in post (DaVinci Resolve or Premiere)
- Specify "slow shutter, motion blur on fast elements" for file movement scenes

### Text overlay fonts (match site):
- Headlines: Newsreader (Google Fonts) — Italic for emotional lines
- Code/terminal: Geist Mono (Vercel) — monospace lines
- All white on dark, centered or bottom-left
