Features

Everything you need

Six powerful commands. Zero configuration required. Works on Windows, macOS, and Linux.

🔍

Scan

Recursively scan directories with size, category, date, and extension filtering. Output as table, JSON, or CSV.

📂

Organize

Auto-sort files into 12 smart categories with configurable naming conventions and duplicate handling.

🧹

Clean

Detect and remove temp files, cache directories, system junk, and logs. Reclaim disk space instantly.

👁️

Watch PRO

Real-time file monitoring with rules engine. Auto-organize new files as they appear in your folders.

↩️

Undo

Full rollback journal for every operation. Safely undo any organization with a single command.

🤖

AI SOP PRO

AI-powered document parsing using Gemini. Generate organization rules from your standard operating procedures.

Usage

Simple by design

No configuration needed. Just point at a folder and go. Everything works out of the box with sensible defaults.

# See what's in a folder
filemayor scan ~/Downloads

# Preview organization
filemayor organize ~/Downloads --dry-run

# Sort everything
filemayor organize ~/Downloads

# Delete junk files
filemayor clean /tmp --yes

# Export as JSON
filemayor scan . --json | jq '.files'
