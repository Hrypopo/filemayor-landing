Usage

Simple by design

No configuration needed. Just point at a folder and go. Everything works out of the box with sensible defaults.

# See what's in a folder
filemayor scan ~/Downloads

# Preview organization
filemayor organize ~/Downloads --dry-run

# Sort everything
filemayor organize ~/Downloads

# Delete junk files
filemayor clean /tmp --yes

# Export as JSON
filemayor scan . --json | jq '.files'
